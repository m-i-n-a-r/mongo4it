DATA ENRICHMENT

I dati sono stati recuperati da OpenData e successivamente arricchiti tramite API esterne. Per fornire un dataset più completo, le informazioni sono state integrate aggiungendo dati in tempo reale.

Roma - Capitale OpenData

Dal sito OpenData di Roma Capitale sono state recuperate le informazioni degli incidenti sul territorio.

Maggiori informazioni

Dati Luce Verde

Sono state integrate le informazioni utilizzando il sito LuceVerde, che fornisce in tempo reale, tramite API, le informazioni relative agli incidenti stradali. In questo modo è possibile sfruttare le potenzialità di Mongo, inserendo nella medesima Collection documenti con informazioni differenti.

Maggiori informazioni

GeoNue

Dal sito GeoNue sono state recuperate le informazioni relative ai Municipi di Roma. Tramite funzioni di Mongo GeoSpatial è stato associato il municipio rispetto alle coordinate dell'incidente.

Maggiori informazioni

RapidApi

Dal sito RapidApi sono state recuperate le informazioni delle condizioni metereologiche degli incidenti estratti in tempo reale.

Maggiori informazioni

Sunset API

Semplici API per reperire l'orario di alba e tramonto.

Maggiori informazioni
